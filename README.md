# Recreación de Netflix 

Clon estático de la interfaz de Netflix construido con HTML y CSS puro.

## Descripción

Proyecto frontend que replica el diseño visual de la página principal de Netflix.
El enfoque principal es la maquetación, el uso de Flexbox y el diseño responsivo.

## Tecnologías utilizadas

![HTML](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)

## Características

- Replica visual de la interfaz principal de Netflix
- Diseño responsivo adaptable a diferentes tamaños de pantalla
- Maquetación con Flexbox
- Sin dependencias ni librerías externas

## Cómo ejecutarlo

No requiere instalación. Solo abre el archivo `index.html` directamente
en tu navegador.
